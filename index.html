Taubyte Showcase

Interactive demo using HTTP functions, Database, Storage, and Messaging.

API base optional

If the API is on a different origin (e.g. Dream: API domain vs Web domain), set it below. Leave empty for same-origin.

Health GET /api/health

HTTP function — health check.

Database — Items GET/POST /api/items

Key-value database (matcher: showcase) via get_items & save_item.

    Storage — File GET/POST /api/file

    Object storage (matcher: showcase_uploads) via get_file & put_file.

    Upload: choose file and optional name. Get: enter name and open or download.

    Messaging & PubSub

    Channel showcaseevents — event_handler runs when messages are published (backend or tools). Not triggered from the browser.
