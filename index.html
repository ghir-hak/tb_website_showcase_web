Taubyte Showcase

This project demonstrates Taubyte resources and features.

HTTP Functions

REST API powered by Taubyte HTTP functions (one per path + method).

  • GET /api/health — Health check
  • GET /api/items — List items (Database)
  • POST /api/items — Save item (Database)
  • GET /api/file?name=... — Get file (Storage)
  • POST /api/file — Upload file (Storage)

Database

Key-value database (matcher: showcase) — used by get_items and save_item.

Storage

Object storage (matcher: showcase_uploads) — used by get_file and put_file. Streaming storage is also configured.

Messaging & PubSub

Messaging resource (channel match: showcaseevents) with an event_handler PubSub function.

Domains

Generated domains for API and Web. Use your API domain to call the endpoints above.
